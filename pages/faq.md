## Troubleshooting

### How do I know which library version I'm using?

The [](cmd:check) command shows software library versions:

- Guild AI
- Python
- TensorFlow
- CUDA and cuDNN

To show this information, run:

``` command
guild check
```

To show more information, use the ``--verbose`` option:

``` command
guild check --verbose
```
