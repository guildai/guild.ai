      census-dnn model

      Wide and deep classifier for census income data using TensorFlow estimators.
      • Operations

      Operations

      cloudml-batch-predict

      Submit a prediction job using a model deployed in Cloud ML.
      Flag Description Default
      instances Instances to use for prediction. prediction-samples.json

      cloudml-deploy

      Deploy a trained classifier in Cloud ML.

      This operation does not have any flags.

      cloudml-hptune

      Tune model hyperparameters in Cloud ML.
      Flag Description Default
      config Hyperparameter tuning configuration. hptuning_config.yaml

      cloudml-predict

      Perform an online prediction using a model deployed in Cloud ML.
      Flag Description Default
      instances Instances to use for prediction. prediction-samples.json

      cloudml-train

      Train the classifier in Cloud ML.

      This operation does not have any flags.

      train

      Train the classifier locally.
      Flag Description Default
      embedding-size Number of embedding dimensions for categorical columns. 8
      epochs Number of training data epochs If both train-steps and epochs are specified, the training job will run for train-steps or epochs, whichever occurs first. If unspecified will run for train-steps. .
      eval-batch-size Batch size for evaluation steps. 40
      eval-steps Number of steps to run evalution for at each checkpoint. 100
      export-format The input format of the exported SavedModel binary Values may be JSON, CSV, or EXAMPLE. . JSON
      first-layer-size Number of nodes in the first layer of the DNN. 100
      layers Number of layers in the DNN. 4
      scale-factor How quickly the size of the layers in the DNN decay. 0.7
      train-batch-size Batch size for training steps. 40
      train-steps Steps to run the training job for. 1000
      verbosity Log level (use DEBUG for more information) Values may be DEBUG, INFO, WARN, ERROR, or FATAL. . INFO
