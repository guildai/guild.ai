      Cloud ML MNIST

      Name cloudml.mnist
      Description MNIST model for Cloud ML
      Version 0.4.0
      Source https://github.com/guildai/index/tree/master/cloudml/mnist
      Maintainer
      • Models
        mnist-cnn

      mnist-cnn model

      CNN classifier for MNIST data.
      • Operations
      • Resources

      Operations

      cloudml-deploy

      Deploy a model to Cloud ML.
      Flag Description Default
      runtime-version TensorFlow runtime version. 1.2

      cloudml-predict

      Send a prediction request to Cloud ML.

      This operation does not have any flags.

      cloudml-train

      Train the classifier in Cloud ML.
      Flag Description Default
      runtime-version TensorFlow runtime version. 1.2

      prepare-data

      Prepare the MNIST data for training.
      Flag Description Default
      validation-size Number of examples to separate from the training data for the validation . 5000

      prepare-request

      Generate a sample request file.

      This operation does not have any flags.

      train

      Train the classifier locally.
      Flag Description Default
      eval-batch-size Batch size for evaluation steps. 100
      eval-steps Number of steps to run evalution for at each checkpoint. 100
      train-batch-size Batch size for training steps. 100
      train-steps Steps to run the training job for. 10000

      Resources

      data

      Prepared MNIST data.
