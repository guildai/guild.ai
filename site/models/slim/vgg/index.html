      TF Slim VGG

      Name slim.vgg
      Description Oxford VGG models (11-Layers verion A, 16-Layers version D, 19-Layers version E)
      Version 0.4.1
      Source https://github.com/guildai/index/tree/master/slim/vgg
      Maintainer
      • Models
        slim-vgg-11 slim-vgg-16 slim-vgg-19

      slim-vgg-11 model

      Oxford VGG 11-Layers version A model.
      • Operations
      • Resources
      • References

      Operations

      evaluate

      Evaluate a trained VGG-11 model.
      Flag Description Default
      dataset Dataset to train with (cifar10, mnist, flowers, custom). required
      max-batches Maximum number of batches to evaluate (default is all).

      export

      Generate a VGG-11 graph def.
      Flag Description Default
      dataset Dataset to train with (cifar10, mnist, flowers, custom). required

      freeze

      Generate a VGG-11 graph def with checkpoint weights.

      This operation does not have any flags.

      predict

      Use TensorFlow label_image and VGG-11 to classify an image.
      Flag Description Default
      dataset Dataset name to use for labels and image transformation. required
      image Path to the input image. required
      input-mean Image mean to apply to the image. 0.0
      input-std Image std deviation to apply to the image. 1.0

      train

      Train a VGG-11 model.
      Flag Description Default
      batch-size Number of samples in each batch. 32
      checkpoint Run ID or path to checkpoint to resume training from.
      dataset Dataset to train with (cifar10, mnist, flowers, custom). required
      learning-rate Initial learning rate. 0.01
      learning-rate-decay-type How the learning rate is decayed. exponential
      log-every-n-steps Steps between status updates. 100
      max-steps Maximum number of training steps. 1000
      optimizer Training optimizer (adadelta, adagrad, adam, ftrl, momentum, sgd, rmsprop). rmsprop
      save-model-secs Seconds between model saves. 60
      save-summaries-secs Seconds between summary saves. 60
      weight-decay Weight decay on the model weights. 4e-05

      Resources

      cifar10

      Prepared CIFAR-10 dataset (slim-cifar10:prepare operation).
      • ['data'] from slim-cifar10:prepare operation

      custom

      Prepared custom dataset (slim-custom:prepare operation).
      • ['data'] from slim-custom-images:prepare operation

      export

      Exported VGG-11 graph def from export operation.
      • ['graph.pb'] from export operation

      flowers

      Prepared Flowers dataset (slim-flowers:prepare operation).
      • ['data'] from slim-flowers:prepare operation

      frozen-model

      Frozen VGG-11 graph with weights.
      • ['graph.pb'] from freeze operation

      mnist

      Prepared MNIST dataset (slim-mnist:prepare operation).
      • ['data'] from slim-mnist:prepare operation

      trained-model

      Trained VGG-11 model (train operation).
      • ['checkpoint|model\\.ckpt.*'] from train or finetune operations

      References

      • GitHub: tensorflow/models/research/slim/nets/vgg.py
      • http://arxiv.org/pdf/1409.1556.pdf
      • http://www.robots.ox.ac.uk/~karen/pdf/ILSVRC_2014.pdf
