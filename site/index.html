      Automate TensorFlow tasks. Manage experiments. Reuse models.

      Guild AI is a TensorFlow task runner and experiment tracker. It automates training, testing, deployment and use of your models.

      Guild projects

      Add guild.yml to your project to automate your TensorFlow model operations and enable experiment tracking, testing, distribution and remote training. Learn more.

      Model operations

      Train your model by running the command guild run train to generate a unique experiment that preserves training files and metadata. Learn more.

      Experiments

      Track experiments to save logs and trained models, compare performance, diff files, visualize results in TensorBoard and backup to the cloud. Learn more.

      End-to-end workflow

      Automate your model's complete life cycle, from data collection and pre-processing to training and evaluation to optimization and deployment. Learn more.

      Model tests

      Verify all stages of your TensorFlow automation workflow with tests that exercise code and check results such as expected loss and accuracy ranges. Learn more.

      Packages and code reuse

      Save time and reduce errors by reusing tested, proven models and operations—developed by you as well as others in the TensorFlow community. Learn more.

      Get started

      Convert a Jupyter Notebook to a Guild project

      If your TensorFlow training code is in a Jupyter Notebook, use Guild to create a reusable project that supports experiment tracking, remote training, and detailed run comparison.

      Convert a Jupyter Notebook

      Add Guild to your TensorFlow or Keras project

      If you have scripts that train your TensorFlow or Keras models, add a Guild file (i.e. a file named guild.yml) to your project to enable Guild features.

      Add Guild to a project
