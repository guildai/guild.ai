      remote status command

      1. Usage
      2. Options

      Usage

      guild remote status [OPTIONS] REMOTE

      Show remote status.

      REMOTE is the name of a configured remote. Use guild remotes to list available remotes.

      For information on configuring remotes, see remotes.

      Options

      -v, --verbose

      Show more status information.
      --help Show command help and exit.
